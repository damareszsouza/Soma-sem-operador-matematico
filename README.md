# Soma-sem-operador-matematico

execução do código abaixo irá escrever o valor 9.8:

programa

{

               inclua biblioteca Util

               funcao inicio()

               {

                               real valores[4] = {1.1,1.2,2.3,3.2}

                               real soma = 2

                               para(inteiro i=0;i<Util.numero_elementos(valores);i++){

                                               se(valores[i]<soma){

                                                               soma+=valores[i]

                                               }

                               }

                               escreva(soma)

               }

}
